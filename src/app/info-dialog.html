How things work! üîß

The Stonk Market is a very primitive approximation of a market. Prices change by some random amount every second or so, but are affected slightly by the buy and sell actions of a user.

The Math Corner is a simple generator that self-checks a random math problem. To make this doable by a human without a calculator, these are limited to values 1-10 and only do addition, subtraction, and multiplication to keep values reasonable.

Nerd stuff! ü§ì

First, this was designed to be both mobile-first, but also mobile-only, so if you're not using a phone, things look pretty silly! This is a standard Angular 10 app and all computation is done in the browser. Hosted on AWS with Cloudfront and Code Pipelines. The Stonks chart is provided by Highcharts using Highstonks Highstocks. Our whimsical font is the delightful "Comic Neue" and the favicon is custom designed by me.
